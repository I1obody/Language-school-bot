# Course Selection & Management Bot

Система автоматизации записи на языковые курсы, реализованная на базе библиотеки Aiogram 3. Бот структурирует процесс сбора заявок, классифицирует предпочтения клиентов и предоставляет инструменты для администрирования базы данных в реальном времени.
Функциональные возможности
Для пользователей

    Сбор контактных данных: Реализован запрос номера телефона через интерфейс Telegram.

    Многоэтапный опрос: Система подбора курса включает выбор языка, формата обучения (очно/онлайн), целей, возрастной категории и уровня владения языком.

    Предотвращение дубликатов: Алгоритм проверяет наличие существующих заявок с идентичными параметрами перед сохранением в базу.

    Информационное сопровождение: Автоматическая отправка координат учебного центра после успешного завершения записи.

Для администрации и владельцев

    Панель администратора: Позволяет просматривать очередь необработанных заявок и изменять их статус непосредственно через интерфейс бота.

    Аналитика и статистика: Генерация отчетов по количеству заявок за 24 часа, неделю, месяц или год.

    Контроль рабочего времени: Логика бота учитывает часовой пояс и график работы персонала. Заявки, поступившие в нерабочее время, помещаются в отдельную очередь.

    Система уведомлений: Реализованы автоматические напоминания о необработанных заявках и сервисные сообщения о начале и завершении рабочего дня.

Для разработчика

    Управление данными: Функционал полной очистки базы данных с механизмом подтверждения.

    Резервное копирование: Создание и выгрузка архива с текущей базой данных в формате .xlsx.

Технологический стек

    Core: Python, Aiogram 3.

    Data Processing: Pandas, Openpyxl.

    Validation: Pydantic.

    Storage: Excel (.xlsx).

    Scheduling: Asyncio.

Логика обработки заявок

Система использует гибридный метод хранения. Текущие заявки записываются в Excel-файл для постоянного хранения. В нерабочее время новые запросы буферизуются в JSON-файл очереди, который обрабатывается и отправляется администратору в виде сводного отчета в начале следующего рабочего дня.
